  1. اذهب لـ github.com/settings/tokens ←

  3. أعطه اسماً مثل waseet-admin
  4. اختر صلاحية repo (كاملة)
  5. اضغط Generate token وانسخ الرمز
  6. الصقه في حقل Token أعلاه واضغط حفظ
